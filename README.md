# 纯净音乐播放器

一个简洁美观的音乐播放器，支持多平台音乐搜索和播放。

## 功能特点

- **多平台搜索**：QQ音乐、网易云音乐、酷狗音乐、酷我音乐
- **播放列表**：自定义播放列表，支持添加/删除/清空
- **播放模式**：列表循环、单曲循环、随机播放
- **歌词显示**：实时滚动歌词，点击歌词行跳转播放
- **主题切换**：支持亮色/暗色主题

## 使用方法

### 1. 启动服务器

```bash
cd gequbao-proxy
npm install
node server.js
```

### 2. 打开播放器

在浏览器中访问：
```
http://localhost:3001
```

### 3. 搜索歌曲

在搜索框输入歌手名或歌曲名，点击搜索按钮。

### 4. 添加到播放列表

点击搜索结果中的 `+` 按钮，将歌曲添加到播放列表。

### 5. 播放模式切换

点击播放控制栏左侧的播放模式按钮：
- 🔁 列表循环
- 🔂 单曲循环
- 🔀 随机播放

## 键盘快捷键

| 按键 | 功能 |
|------|------|
| Space | 播放/暂停 |
| ← | 快退10秒 |
| → | 快进10秒 |
| ↑ | 音量增加 |
| ↓ | 音量减少 |
